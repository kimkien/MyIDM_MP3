# MyIDM_MP3
Bắt gói tin HTTP và phân tích thành phần để tìm link mp3 và tải về local
 1)  Chạy ứng dụng:
    chạy file WinPcap_Demo.cpp
 2) Nội dung khai thác:
  Gốm 2 phần chính:
    a) Sử dụng winpcap để bắt gói tin
    b) Sử dụng winsock để tạo socket gửi request và tải về file mp3 (blocking và non-blocking)
